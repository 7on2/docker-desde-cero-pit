Gestor de Tareas

Aplicacion Flask conectada a PostgreSQL dentro de Docker.

    {% for tarea in tareas %}
  • #{{ tarea[0] }} - {{ tarea[1] }} {% if tarea[2] %}(completada){% endif %}

Crear tarea:

curl -X POST http://localhost/tareas -H 'Content-Type: application/json' -d '{"titulo":"Nueva tarea"}'
